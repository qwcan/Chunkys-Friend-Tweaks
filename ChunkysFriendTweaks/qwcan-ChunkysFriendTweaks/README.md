# Chunky's Friend Tweaks
Tweaks to LethalLib and LethalLevelLoader, intended for my friend group. Allows for blacklisting scrap or tweaking their rarities and values. I made this when we didn't get the results we wanted with much more complicated mods like LunarConfig or LethalCompanyVariables.

Source: https://github.com/qwcan/Chunkys-Friend-Tweaks
